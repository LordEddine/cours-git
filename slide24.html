Modifier le dernier commit

Corriger une erreur dans le commit précédent sans créer un nouveau commit.

git commit --amend

# Modifier le message du dernier commit
$ git commit --amend -m "Nouveau message"

# Ajouter des fichiers oubliés
$ git add fichier_oublie.py
$ git commit --amend --no-edit
Ajoute sans changer le message

⚠️ Attention

Jamais après un push !

Amend réécrit l'historique. Si déjà pushé, cela créera des conflits.

  • ✅ Commit local uniquement
  • ✅ Avant de partager
  • ✗ Après git push
  • ✗ Branches partagées
