EXERCICE 3

Collaboration sur GitHub

Instructions

  1. 1 Créer un dépôt python-utils sur GitHub
  2. 2 Lier votre projet local au remote
  3. 3 Créer utils.py avec quelques fonctions
  4. 4 Push vers GitHub
  5. 5 Créer une branche, modifier, et CRÉER UNE PR
  6. 6 Merger la PR sur GitHub

Commandes utiles

# Configurer le remote
$ git remote add origin [URL]

# Premier push
$ git push -u origin main

# Push une branche
$ git push -u origin feature
