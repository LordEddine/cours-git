Dépôts distants (remotes)

Un remote est une version de votre projet hébergée sur Internet (GitHub, GitLab...).

Configurer un remote

$ git remote add origin [URL]
Ajouter un remote nommé "origin"
$ git remote -v
Voir les remotes configurés
$ git remote remove origin
Supprimer un remote

Synchroniser

$ git push -u origin main
Envoyer + définir upstream
$ git pull origin main
Récupérer + fusionner
$ git fetch origin
Récupérer sans fusionner
